# duckdb_sakila_handler

A data analysis project based on the Sakila DVD Rental database.
The repo contains:

- Data ingestion from SQLite to DuckDB

- Exploratory data analysis (EDA) using Pandas and DuckDB

- Visualizations for customer spending and film category revenue

- A BI dashboard built with Evidence for presenting insights
  Notebooks, code, and documentation are included for reproducibility.
